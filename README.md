# AI-project

A simple Tic-Tac-Toe AI project that uses the Minimax algorithm.

## Description

This project includes a Python game where a human player plays as `X` and an AI uses the Minimax algorithm to play as `O`.

## Files

- `tic_tac_toe.py` - Main game implementation with Minimax AI.

## Run the Python game

1. Make sure you have Python 3 installed.
2. Run:

```bash
python3 tic_tac_toe.py
```

3. Enter a move from `1` to `9` when prompted.

## Run the web frontend

1. Open `index.html` in your browser.
2. Click any empty cell to place your `X`.
3. The AI will respond automatically as `O`.
